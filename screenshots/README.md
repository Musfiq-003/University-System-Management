# Screenshots for the Report

This folder is referenced by `DATABASE_ARCHITECTURE_REPORT.md`.

## Required filenames

Save your screenshots using these exact names so they render in the report:

- `login.png`
- `dashboard.png`
- `routine.png`
- `research-paper.png`
- `hostel.png`
- `admin-panel.png`
- `activity-logs.png`

## How to capture

- Run backend: `npm start` (port 3000)
- Run frontend: `cd frontend` then `npm start` (port 3001)
- Open each page, take a screenshot, and save it here using the filenames above.

Tip: PNG is preferred for clarity.
